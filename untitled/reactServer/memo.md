## 설치 cli
### react-router
- 리액트에서 이동한 페이지의 내용을 출력하기 위해서 필요
```bash
npm install react-router-dom
```

### yarn
- 패키지 매니저로 yarn을 사용하기 위해 필요
```bash
npm install -g yarn
```

### axios
- 통신을 위해 필요
```bash
yarn add axios
```

### yarn 실행
```bash
yarn start
```
